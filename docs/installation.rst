============
Installation
============


Stable release
--------------

To install git_log_analysis, run this command in your terminal:

.. code-block:: console

    $ pip install git_log_analysis

This is the preferred method to install git_log_analysis, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for git_log_analysis can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/geongupark/git_log_analysis

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/geongupark/git_log_analysis/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/geongupark/git_log_analysis
.. _tarball: https://github.com/geongupark/git_log_analysis/tarball/master
